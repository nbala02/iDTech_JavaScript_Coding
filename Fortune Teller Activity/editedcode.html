GitHub Install

If you haven't already, create an account on github.com. Follow along with this video to download git, if you haven't yet, and set it up on your local machine using the ssh keys. You can also review the documentation on how to set up ssh keys on your GitHub account.

 Mac Users:

If you're on a Mac computer, your instructions may differ a bit from the video below. While watching the video, you can follow along with the written steps below the video.

 Windows Users:

If you're on a Windows computer, your instructions may differ a bit from the video below. Once you reach 2:53 in the video, you can follow along with the written steps in the "ssh Keys on Windows" chunk.

ssh Keys on Mac

  1. At timestamp 1:48, you'll be lead to this page, to install git open up Terminal.
  2. In Terminal, run the following command to install git on your Mac.
    • $ brew install git
    ⚡️ Homebrew Error Message

    If you run into an error or don't have homebrew installed on your computer, paste the following line of code in Terminal and press enter:

    • /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    You'll need to enter your computer's password and press the return key to begin the installation process.

  3. At timestamp 2:32, access the .ssh directory by typing:
    • cd .ssh
  4. At timestamp 3:23, instead of entering a file path for the key, keep the default location and press enter.
  5. Press enter for the next 3 prompts about the location and passphrase.

  6. At timestamp 3:58, you'll list the data from the default id_ed25519 file instead of the test1 file.

ssh Keys on Windows

As you may have noticed in the documentation, the steps to generate your ssh keys will differ if you're on Windows. These steps can all be done in the Git Bash terminal.

  1. Open Git Bash and type the following command to enter the .ssh folder on your computer:
    • cd .ssh

  2. Type the following command to generate a key for your GitHub email address (where you'll replace "YourName@YourEmail.com" with your GitHub login email address):
    • ssh-keygen -t ed25519 -C "YourName@YourEmail.com"

  3. Press enter for the next 3 prompts about the location and passphrase.

  4. To make sure the Git Agent is running, type:
    • eval "$(ssh-agent -s)"

  5. To add the key to the Git Agent, type:
    • ssh-add

  6. Copy the ssh key to your clip board by typing:
    • clip < ~/.ssh/id_ed25519.pub

    If it doesn't properly copy to your clipboard for some reason, you can also go to the “C:\Users\”UsernameHere”\.ssh” folder on your computer and open the “id_ed25519.pub” file in VS Code.

  7. Scroll back up to the video and continue watching at 6:32 to add the copied key to your GitHub account.
